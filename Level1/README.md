# Level 1 Projects

## Temperature Converter
Converts temperatures between Celsius and Fahrenheit.

## Palindrome Checker
Checks whether a word or phrase is a palindrome (ignoring spaces and punctuation).

## Student Grade Calculator
Calculates the average grade from a list of grades.

## Random Password Generator
Generates a random password based on user-specified options (length, lowercase, uppercase, digits, special characters).

## How to Run
```powershell
javac TemperatureConverter.java
java TemperatureConverter
```
