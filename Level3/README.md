# Level 3 Projects

## Chat Application (Networking)
Client-server chat application using sockets. Run `ChatServer` in one terminal and `ChatClient` in another.

## Multi-threaded Application
Demonstrates concurrency using two threads incrementing and decrementing a shared counter with `ReentrantLock`.

## Desktop Application (Swing)
A simple calculator GUI built with Java Swing.

## Currency Converter
Converts currency using live exchange rates from exchangerate-api.com.

## How to Run
```powershell
javac ChatServer.java
javac ChatClient.java
java ChatServer    # Terminal 1
java ChatClient    # Terminal 2
```
