# Level 2 Projects

## Tic-Tac-Toe Game
Two-player console tic-tac-toe game with win/draw detection and replay option.

## Password Strength Checker
Analyzes a password's strength based on length, character variety, and special characters.

## File Encryption/Decryption
Encrypts or decrypts text files using a Caesar cipher (shift of 3).

## How to Run
```powershell
javac TicTacToe.java
java TicTacToe
```
