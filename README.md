# Cognifyz Java Development Internship

Java projects for the Cognifyz internship program.

## Compilation
From the project root, run:
```powershell
javac Level1\*.java
javac Level2\*.java
javac Level3\*.java
```

## Running
```powershell
# Level 1
java Level1.TemperatureConverter
java Level1.PalindromeChecker
java Level1.StudentGradeCalculator
java Level1.RandomPasswordGenerator

# Level 2
java Level2.TicTacToe
java Level2.PasswordStrengthChecker
java Level2.FileEncryptionDecryption

# Level 3
java Level3.ChatServer    # Terminal 1
java Level3.ChatClient    # Terminal 2
java Level3.MultiThreadedApplication
java Level3.DesktopApplication
java Level3.CurrencyConverter
```
